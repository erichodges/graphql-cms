### Contributing

**Fork this repo.**

**Clone the repo.**
```sh
git clone git@github.com:YOUR_USERNAME/graphql-cms.git
```

**CD into the clone.**
```
cd graphql-cms
```

**Install dependencies.**
```
yarn # or npm i
```

**Start webpack.**
```sh
yarn start
```

Go to `localhost:8080` now!

Please limit PRs to a single fix or feature.
